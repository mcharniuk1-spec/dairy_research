# FINAL_RESEARCH Materials

This folder keeps the external files that the final rerun depends on, so `FINAL_RESEARCH` remains self-contained.

## Structure
- `inputs/`: source workbooks and the final modelling workbook used directly by the pipeline.
- `references/`: thesis draft and writing references used to shape Chapter 5 and Chapter 6.
- `instructions/`: methodology notes and correction documents that informed the rebuild.

## Main items
- `inputs/full_uah_final.xlsx`: corrected governmental source workbook.
- `inputs/full_uah_final_whatadded_matched_smoothed.xlsx`: final matched and smoothed modelling workbook used by the pipeline.
- `references/Charniuk_Maksym_MScThesis_Draft_correctedformat.docx`: master thesis draft used as the style reference.
